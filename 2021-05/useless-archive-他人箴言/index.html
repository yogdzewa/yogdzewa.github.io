计算机科学

计算机科学不是培养程序员的学科,是我们要找工作所以需要学编程 :

  1. 什么是计算
  2. ->怎么造
  3. ->怎样高效计算 programming(按一定顺序执行特定步骤, 不同于coding, 原本是一个数学概念)
  4. 针对特定问题的计算方法(algorithm), 如数据结构等
  5. 如何解决现实问题 application
信息文明

许多现实问题都已经转化为计算问题(工业文明大部分都转化为发动机问题)

4+2

  2. 学习编码是为了能让程序运行的更快更稳定coding

各种段位: (五等)

20行 -> 200行 -> 2000行(找工作不愁) -> 2 0000行(可以砍价要股份) -> 20 0000行(基本不可能, 如Linux的编写)

代码能力成熟度:(3等)普工

  1. 当时能跑,明天起来就不记得了,不如重写
  2. 有文档有注释, 但只有自己能看懂, 半年后回来还是要重写
  3. 别人能搞懂, 别人能维护, 代码可靠, 能被重用, 能在此基础上做二次开发

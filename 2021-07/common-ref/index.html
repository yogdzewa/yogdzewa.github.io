REP & STOS

INSTRUCTION DESCRIPTION
REP MOVS m8, m8 Move RCX bytes from [RSI] to [RDI]
REP STOS m8 Fill RCX bytes at [RDI] with AL.
REP STOS m16 Fill (E)CX words at ES:[(E)DI] with AX.

括号的意思是使用时用的是双字寄存器, 但是具体数值长度由前面几句汇编装载使用的大小来决定.

The REP (repeat), REPE (repeat while equal), REPNE (repeat while not equal), REPZ(repeat while zero), and REPNZ (repeat while not zero)

Repeat Prefix Termination Condition 1 Termination Condition 2
REP RCX or (E)CX = 0 None
REPE/REPZ RCX or (E)CX = 0 ZF = 0
REPNE/REPNZ RCX or (E)CX = 0 ZF = 1

the REPE, REPNE (sync: REPZ, REPNZ ) prefixes can be added to the CMPS and SCAS instructions.

FILE-objects: more easy to use; built-in-buffering;

fd-objects: more close to operating system control;

stdin, stdout and stderr are all FILE pointers, which are contain buffers.

    • * means ignore.

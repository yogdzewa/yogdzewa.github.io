REP & STOS

INSTRUCTION DESCRIPTION
REP MOVS m8, m8 Move RCX bytes from [RSI] to [RDI]
REP STOS m8 Fill RCX bytes at [RDI] with AL.
REP STOS m16 Fill (E)CX words at ES:[(E)DI] with AX.

括号的意思是使用时用的是双字寄存器, 但是具体数值长度由前面几句汇编装载使用的大小来决定.

The REP (repeat), REPE (repeat while equal), REPNE (repeat while not equal), REPZ(repeat while zero), and REPNZ (repeat while not zero)

Repeat Prefix Termination Condition 1 Termination Condition 2
REP RCX or (E)CX = 0 None
REPE/REPZ RCX or (E)CX = 0 ZF = 0
REPNE/REPNZ RCX or (E)CX = 0 ZF = 1

the REPE, REPNE (sync: REPZ, REPNZ ) prefixes can be added to the CMPS and SCAS instructions.

c file I/O

small tip about fopen() and open(): (FILE is an abstrace layer over os-provided fd. )

FILE-objects: more easy to use; built-in-buffering;

fd-objects: more close to operating system control;

stdin, stdout and stderr are all FILE pointers, which are contain buffers.

scanf format consists of a sequence of directives.

  • white-space: match ANY amount of white-space(0-∞).
  • ordinary chars.
  • conversion specification: commence with ‘%’: %[*/m/n$][width][type_modifiers(l)]specifier(d)
    • m means scanf allocate a memory for receive string.
      run-c 'char* buf; scanf("%*c %ms", &buf); puts(buf);' input: a abc output: abc
    • * means ignore.

printf format consists of two kinds of directives: ordinary chars, conversion specifications, which ends with conversion specifier:
%[$][flags][width][.precision][length modifier]conversion

  • flags: C99#(alternative form) 0(zero padded) -(left_adjust) ' ' +(about sign conversion) UNIX'(grouping) glibcI(locale)
  • width: field won’t be truncated due to nonexistence or small width.
  • precision: 没啥特别的. 最少整数数字, 小数精度, 字符串最多输出字符
  • length: 用时在看
  • conversion: 浮点数有直接和指数两种表示方式. 数字可以限制进制.
